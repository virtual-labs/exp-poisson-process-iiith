Instructions
  • In this experiment, you will visualize—on the same plot—the Binomial PMF arising from \(n\) repeated Bernoulli trials with success probability \(p = \tfrac{\lambda}{n}\), and the Poisson PMF with mean \(\lambda\). By increasing \(n\) (the number of trials), while keeping \(\lambda\) fixed, you will see how the Binomial\(\bigl(n,\,\tfrac{\lambda}{n}\bigr)\) distribution converges to a Poisson\((\lambda)\).
  • Use the sliders below to choose:
    1. \(n\) (the number of Bernoulli trials): as \(n\) becomes large, \(p = \tfrac{\lambda}{n}\) becomes small, but the mean remains \(\lambda\).
    2. \(\lambda\) (the Poisson mean): this also determines the expected number of successes in the Binomial model.
  • For the selected \((n, \lambda)\):
    1. Compute the Binomial PMF \[ P_{\text{Bin}}(k) \;=\; \binom{n}{k} \Bigl(\tfrac{\lambda}{n}\Bigr)^{k} \Bigl(1 - \tfrac{\lambda}{n}\Bigr)^{\,n - k},\quad k = 0,1,\ldots, K_{\max}, \] where \(K_{\max} = \min\bigl(n,\;\lceil \lambda + 4\sqrt{\lambda}\rceil\bigr)\).
    2. Compute the Poisson PMF \[ P_{\text{Pois}}(k) \;=\; e^{-\lambda} \frac{\lambda^{\,k}}{k!},\quad k = 0,1,\ldots, K_{\max}. \]
    3. Plot both PMFs on the same set of axes (Binomial in red, Poisson in blue).
    4. The panel below the plot will show the average absolute error \(\displaystyle \frac{1}{K_{\max}+1}\sum_{k=0}^{K_{\max}} \bigl|P_{\text{Bin}}(k) - P_{\text{Pois}}(k)\bigr|\).
  • Observe how, as \(n\) grows, the two curves align more closely and the average error shrinks.
